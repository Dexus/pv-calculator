PVGIS Mobile Import – PV + Speicher + 800 W WR

Workaround ohne lokalen Proxy: Diese HTML-App erzeugt PVGIS-Links, du öffnest/speicherst/kopierst die PVGIS-Daten im Browser und importierst sie danach. Die Batterie-, SOC-Carry-over- und 24h-800-W-Simulation läuft vollständig lokal im Browser.

3) PVGIS-Links für Handy-Workaround

Ablauf auf dem Handy: Link öffnen → PVGIS JSON anzeigen oder speichern → zurück zur App → Datei importieren oder JSON in das Textfeld einfügen. Das ist der saubere Workaround ohne lokalen Proxy.
